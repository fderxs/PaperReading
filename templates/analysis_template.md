## 1. 一句话结论

用 2 到 4 句话回答：这篇论文到底解决什么问题，为什么值得或不值得继续读。

## 2. 论文定位

| 维度 | 判断 |
|---|---|
| 方向 | `VLA / Planning / World Model / Dataset / Benchmark / Survey / Other` |
| 应用场景 | 例如机器人操作、医疗机器人、自动驾驶、工业部署等 |
| 方法类型 | 例如模型架构、训练策略、校准方法、规划算法、数据集/基准等 |
| 与我当前兴趣的关系 | 说明它为什么匹配或不匹配 VLA / 具身规划 |

## 3. 背景与动机

说明作者认为现有方法有什么问题。建议按“已有方法 -> 痛点 -> 本文切入点”来写。

## 4. 核心贡献

用 3 到 5 条列出真正的贡献，不要照搬论文摘要。每条都要说明“新在哪里”和“解决了什么问题”。
